
<div align="center"> <h1>Flippr</h1> <p>A seamless, client-side multimedia converter powered by FFmpeg.wasm for fast and free file transformations.</p> </div>

<hr>

<h2>Core Functionality</h2> <ul> <li><strong>Universal Multimedia Conversion:</strong> Effortlessly transforms files across three major categories: Images (JPG, PNG, WebP, SVG, etc.), Video (MP4, MKV, AVI, 3GP, etc.), and Audio (MP3, WAV, FLAC, etc.).</li> <li><strong>Client-Side Processing:</strong> Leverages <code>@ffmpeg/ffmpeg</code> to execute heavy-duty media operations directly in the user's browser, ensuring data privacy and reducing server latency.</li> <li><strong>Intelligent Format Mapping:</strong> Features a dynamic UI that automatically filters available output formats based on the detected input file type (e.g., offering audio extraction from video files).</li> <li><strong>Bulk Action Support:</strong> Designed with a multi-file queue system that allows users to upload, configure, and convert entire batches of media simultaneously.</li> <li><strong>Mobile-Optimized Presets:</strong> Includes specific encoding logic for legacy mobile formats, such as 3GP, with pre-configured resolution and bitrate settings.</li> </ul>

<h2>Technical Architecture</h2> <ul> <li><strong>Core Engine:</strong> Built with <strong>Next.js 14</strong> and <strong>TypeScript</strong> for a robust, type-safe development environment.</li> <li><strong>WASM Integration:</strong> Utilizes WebAssembly to run the FFmpeg binary in-browser, enabling native-speed media processing without external software installation.</li> <li><strong>State Management:</strong> Implements React hooks and refs (<code>useState</code>, <code>useEffect</code>, <code>useRef</code>) to manage complex conversion lifecycles from upload to download.</li> <li><strong>UI/UX:</strong> Crafted with <strong>Tailwind CSS</strong> and <strong>Shadcn UI</strong> components, featuring a drag-and-drop interface, real-time conversion progress badges, and accessible theme toggles.</li> </ul>
<h2>Usage Guide</h2> <ol> <li><strong>Upload:</strong> Drag and drop your images, videos, or audio files into the designated dropzone.</li> <li><strong>Configure:</strong> For each file, select the desired target format from the "Convert to" dropdown menu.</li> <li><strong>Process:</strong> Click "Convert Now" to trigger the local FFmpeg pipeline; observe the status badges for real-time updates.</li> <li><strong>Download:</strong> Once converted, download files individually or use the "Download All" feature to save your processed media.</li> </ol>
